# Phone Logger

## Description
A utility application for logging phone activity and communications.

## Features
- Activity logging
- Call history tracking
- Message archival
- Timestamp records

## Installation
```bash
git clone <repository-url>
cd phone-logger
```

## Usage
```bash
python phone_logger.py
```

## Configuration
Update `config.json` with your settings.

## Requirements
- Python 3.8+
- See `requirements.txt` for dependencies
